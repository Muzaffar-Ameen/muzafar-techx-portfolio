# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Muzaffar-Ameen/muzafar-techx-portfolio.git
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

# 🚀 Build Project

```bash
npm run build
```
